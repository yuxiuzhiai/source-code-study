MetadataReaderFactory

接口注释

    MetadataReader的工厂接口

接口定义
    
    public interface MetadataReaderFactory {

        MetadataReader getMetadataReader(String className) throws IOException;

        MetadataReader getMetadataReader(Resource resource) throws IOException;

    }

子类

    SimpleMetadataReaderFactory:
        为每个请求创建一个org.springframwork.asm.ClassReader

    CachingMetadataReaderFactory:
        继承自SimpleMetadataReaderFactory,为每个Resource提供缓存

    ConcurrentReferenceCachingMetadataReaderFactory:
        继承自SimpleMetadataReaderFactory,也是提供缓存,不过是缓存在ConcurrentReferenceHashMap中
